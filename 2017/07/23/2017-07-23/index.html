目标

  1. 修改网站模板
  2. 稳定网站上传

问题的解决过程

网站每上传一次，绑定的域名必掉，需要重新登陆Github重新绑定！

  1. 网站的本地构建没有问题
  2. 网站可以上传数据至GitHub，不确定是否是网站的部署问题
  3. 网站域名绑定与CNAME文件相关。
  4. 本地的 public文件夹中无CNAME文件
  5. 在本地public文件中新建CNAME文件，上传文件，部署网站。
  6. 问题得到解决，无再次出现此现象
