这是一个我的个人博客小站

已将以前旧网站上的绝大多数内容都搬到了这里

旧的站点依然还可以访问，不过以后不会再更新了。会把发布新内容的使命转移到这个网站上。

这个网站也和旧站一样是个静态网页网站，文档均为Markdown格式。

小站成现运用到的工具有Github，Node.js，Git，Hexo，Typoa，chrome。

  1. Github Pages 实现网页托管

  2. Hexo 博客框架实现网站的构建

  3. Godaddy 买的域名

  4. 百度云加速提供域名解析 已更换dnspod（腾讯云）解析 更换cloudflare.com 解析以及CDN加速。

  5. 七牛云存储提供图床 更换https://postimages.org/ 图床服务。（为了实现网站小绿锁）

唯一花钱的是买域名。
